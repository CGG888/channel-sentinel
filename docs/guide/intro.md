# 介绍

Channel Sentinel 是一款专业的 **IPTV 频道检测与回放规则管理工具**，帮助用户高效管理和测试 IPTV 直播源。

## 核心特性

### 🔍 智能频道检测

- 支持标准 M3U/M3U8 格式解析
- 自动识别直播源类型（单播、组播）
- 多协议支持：HTTP、HTTPS、RTSP、RTMP 等
- 实时检测频道可用性和延迟

### 📺 回放规则引擎

- 基于 JSON 的规则配置系统
- 支持省份、运营商、城市多维度匹配
- 15+ 种时间格式自动识别
- 规则热更新，无需重启服务

### 🌐 社区驱动

- GitHub OAuth 安全认证
- 规则贡献与分享平台
- 完整的贡献者溯源机制
- 规则版本管理与回滚


## 适用场景

| 场景 | 说明 |
|------|------|
| IPTV 爱好者 | 管理私人直播源列表，测试频道可用性 |
