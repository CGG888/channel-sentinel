## 高级功能

### 并发控制

调整并发数可以加快检测速度，但可能影响网络：

```javascript
// config.json
{
  "checker": {
    "maxConcurrency": 10  // 最大并发数
  }
}
```

### 规则调试

启用调试模式查看详细日志：

1. 打开 **回放规则** 面板
2. 勾选 **调试模式**
3. 操作后查看 **命中日志**

### 自定义规则

#### 添加省份规则

```json
{
  "id": "custom-rule-1",
  "enabled": true,
  "priority": 100,
  "region": {
    "province": "广东",
    "operator": "电信",
    "city": "*"
  },
  "match": {
    "stream_type": "unicast",
    "protocols": ["http", "rtsp"]
  },
  "transform": {
    "output_template": "{live_url}"
  }
}
```

## 常见问题

### Q: 检测超时怎么办？

A: 调整 `config.json` 中的超时设置：

```json
{
  "checker": {
    "timeout": 30000  // 30秒超时
  }
}
```

### Q: 如何导出结果？

A: 在结果页面点击 **导出** 按钮，支持 JSON/CSV 格式。
